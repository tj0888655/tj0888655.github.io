Wireframes and Responsive design

We jumped into the pre-planning of web design in the form of wireframes, understanding how a Web Designer can work better when given a visual represntation of what a client might be looking for in their website. We utilized Figma.com for these Wireframes.

After understanding how stylesheets are utilized we then jump over how a user might be interacting with our websites, whether it be on desktop or mobile. We looked into how we could solve this by using media queries that change the website based on the size of the screen.

Wireframes really helped in making decisions on how the site should look like at smaller sizes. One was used to make this website.
